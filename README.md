## התקנה — פעם אחת (Windows)

### 1. התקן Python
מ-https://python.org — סמן **"Add Python to PATH"** בהתקנה.

### 2. התקן ספריות
פתח שורת פקודה בתיקיית הפרויקט והרץ:
```
pip install feedparser requests
```

### 3. הגדר מייל
- העתק את `config/email_config.example.json` ל-`config/email_config.json`
- ב-Gmail: היכנס ל-https://myaccount.google.com/apppasswords וצור **"סיסמת אפליקציה"**
- הדבק אותה בשדה `app_password`, ומלא `from_email` ו-`to_email` בכתובת שלך
- ⚠️ הסיסמה נשארת רק אצלך בקובץ הזה. לא בקוד, לא אצל אף אחד.

### 4. הקם את האתר באינטרנט (GitHub Pages) — כדי שהקישור בהודעות יעבוד
המאגר כבר מוכן מקומית (git init + commit ראשון בוצעו). מה שנשאר לך:
1. היכנס ל-https://github.com/new וצור מאגר (repo) חדש:
   - שם: `haboker-sheli` (או כל שם אחר — רק תעדכן את `site_url` בהתאם)
   - **Public** (נדרש לאחסון חינמי ב-GitHub Pages)
   - **אל** תסמן "Add a README" — יש לנו כבר קבצים מוכנים.
2. בתיקיית הפרויקט, הרץ (פעם אחת):
   ```
   git remote add origin https://github.com/bitonorel9-alt/haboker-sheli.git
   git push -u origin main
   ```
   ייתכן שיופיע חלון דפדפן להתחברות ל-GitHub — אשר אותו.
3. ב-GitHub: **Settings → Pages** → Source: "Deploy from a branch" → Branch: `main` / `(root)` → Save.
   תוך דקה-שתיים האתר יהיה זמין ב: `https://bitonorel9-alt.github.io/haboker-sheli/`
4. ב-GitHub: **Settings → Actions → General → Workflow permissions** → בחר
   **"Read and write permissions"** → Save. (בלי זה, ה-Action היומי לא יוכל לשמור עדכונים).
5. הקישור כבר מוכן וממתין בקובץ `config/whatsapp_config.json` (שדה `site_url`) —
   אם שינית שם משתמש/מאגר, עדכן אותו שם.

מרגע זה, `.github/workflows/update-news.yml` ירוץ אוטומטית כל בוקר ב-Actions
של GitHub (גם אם המחשב שלך כבוי!), יאסוף חדשות טריות, ויעדכן את האתר החי.
לבדיקה מיידית בלי לחכות למחר: לשונית **Actions** במאגר → "עדכון חדשות יומי" → **Run workflow**.

### 5. הגדר וואטסאפ (חינמי, דרך CallMeBot)
- שמור את המספר **+34 644 59 71 07** באנשי הקשר שלך בוואטסאפ.
- שלח לו הודעה בדיוק כך: `I allow callmebot to send me messages`
- תוך דקה-שתיים יגיע אליך מפתח API בהודעת תשובה.
- פתח את `config/whatsapp_config.json` (כבר קיים, עם המספר שלך ממולא) ומלא את `apikey` שקיבלת.
- אם תדלג על השלב הזה — הכול ימשיך לעבוד, פשוט לא תקבל וואטסאפ (רק מייל).

### 6. בדוק שהכול עובד
```
python scripts\collect_news.py
python scripts\send_email.py
python scripts\send_whatsapp.py
```
אמורים להגיע מייל **וגם** הודעת וואטסאפ עם קישור לאתר החי, ופתיחת `index.html` תציג את חדשות היום.

### 7. תזמן ל-07:30 כל בוקר (מחשב מקומי — למייל/וואטסאפ)
לחיצה כפולה על `setup_windows_task.bat` (אשר הרשאות מנהל אם יבקש).
זה שולח את המייל/וואטסאפ; **בתנאי שהמחשב דלוק** באותו רגע.
(עדכון האתר עצמו קורה בענן דרך GitHub Actions, גם אם המחשב כבוי — ראו סעיף 4).

### 8. התקן כאפליקציה (PWA)
- **באייפון:** פתח את הקישור `https://bitonorel9-alt.github.io/haboker-sheli/` בספארי →
  כפתור השיתוף (הריבוע עם החץ) → **"הוסף למסך הבית"**.
  יופיע אייקון "הבוקר שלי" שנפתח כמו אפליקציה, בלי שורת כתובת.
- **באנדרואיד/כרום:** פתח את הקישור → תפריט (⋮) → **"התקן אפליקציה"** / "הוסף למסך הבית".
- ה-Service Worker (`sw.js`) שומר קאש כדי שהאפליקציה תיפתח גם כשאין אינטרנט, עם הנתונים מהפעם האחרונה.
