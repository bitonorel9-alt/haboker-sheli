# ☀️ הבוקר שלי — תמצית חדשות יומית מותאמת אישית

אתר חדשות מודרני שמותקן כאפליקציה (PWA) + מנוע איסוף יומי + שליחת מייל **וגם וואטסאפ** כל בוקר ב-07:30.
נבנה עבורך: כדורגל (ישראל+עולם), פוליטיקה (ישראל), מזג אוויר (עתלית), AI, בידור, ונושא יומי חדש ללמוד + עובדה.

## מה יש כאן
| קובץ | תפקיד |
|------|-------|
| `index.html` | האתר עצמו — פותחים בדפדפן או מתקינים כאפליקציה. כתבות קצרות, סטטיסטיקות בלחיצה, דירוג 👍/👎 |
| `manifest.json` + `sw.js` + `icons/` | הופכים את האתר לאפליקציה מותקנת (PWA) עם אייקון במסך הבית |

## התקנה — פעם אחת (Windows)

### 1. התקן Python
מ-https://python.org — סמן **"Add Python to PATH"** בהתקנה.

### 2. התקן ספריות
פתח שורת פקודה בתיקיית הפרויקט והרץ:
```
pip install feedparser requests
```

### 3. הגדר מייל
- העתק את `config/email_config.example.json` ל-`config/email_config.json`
- ב-Gmail: היכנס ל-https://myaccount.google.com/apppasswords וצור **"סיסמת אפליקציה"**
- הדבק אותה בשדה `app_password`, ומלא `from_email` ו-`to_email` בכתובת שלך
- ⚠️ הסיסמה נשארת רק אצלך בקובץ הזה. לא בקוד, לא אצל אף אחד.

### 4. הגדר וואטסאפ (חינמי, דרך CallMeBot)
- שמור את המספר **+34 644 59 71 07** באנשי הקשר שלך בוואטסאפ.
- שלח לו הודעה בדיוק כך: `I allow callmebot to send me messages`
- תוך דקה-שתיים יגיע אליך מפתח API בהודעת תשובה.
- העתק את `config/whatsapp_config.example.json` ל-`config/whatsapp_config.json`
- מלא `phone` (עם קידומת מדינה, למשל `972501234567`) ו-`apikey` שקיבלת.
- אם תדלג על השלב הזה — הכול ימשיך לעבוד, פשוט לא תקבל וואטסאפ (רק מייל).

### 5. בדוק שהכול עובד
```
python scripts\collect_news.py
python scripts\send_email.py
python scripts\send_whatsapp.py
```
אמורים להגיע מייל **וגם** הודעת וואטסאפ, ופתיחת `index.html` תציג את חדשות היום.

### 6. תזמן ל-07:30 כל בוקר
לחיצה כפולה על `setup_windows_task.bat` (אשר הרשאות מנהל אם יבקש).
מעכשיו המחשב יריץ הכול אוטומטית כל בוקר — **בתנאי שהמחשב דלוק**.

### 7. התקן כאפליקציה (PWA)
- **באייפון:** פתח את `index.html` בספארי → כפתור השיתוף (הריבוע עם החץ) → **"הוסף למסך הבית"**.
  יופיע אייקון "הבוקר שלי" שנפתח כמו אפליקציה, בלי שורת כתובת.
- **באנדרואיד/כרום:** פתח את האתר → תפריט (⋮) → **"התקן אפליקציה"** / "הוסף למסך הבית".
- מכיוון שאין כרגע אחסון אונליין, כדי שהאייקון יעבוד גם באייפון צריך לפתוח את `index.html`
  ממקום נגיש לטלפון (למשל תיקיית iCloud/Drive משותפת, או פריסה לאתר חינמי כמו Netlify —
  ראו "הרחבות אפשריות" ב-`CLAUDE.md`). ה-Service Worker (`sw.js`) שומר קאש כדי שהאפליקציה
  תיפתח גם כשאין אינטרנט, עם הנתונים מהפעם האחרונה.
